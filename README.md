# Group Expiration

![License](https://img.shields.io/badge/license-MIT-blue.svg) [![Latest Stable Version](https://img.shields.io/packagist/v/hertz-dev/flarum-group-expiration.svg)](https://packagist.org/packages/hertz-dev/flarum-group-expiration) [![Total Downloads](https://img.shields.io/packagist/dt/hertz-dev/flarum-group-expiration.svg)](https://packagist.org/packages/hertz-dev/flarum-group-expiration)

A [Flarum](https://flarum.org) extension. Set expiration dates for user groups.

## Installation

Install with composer:

```sh
composer require hertz-dev/flarum-group-expiration:"*"
```

## Updating

```sh
composer update hertz-dev/flarum-group-expiration:"*"
php flarum migrate
php flarum cache:clear
```
